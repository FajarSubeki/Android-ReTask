# Android-ReTask

  ReTask atau Reminder Task Merupakan aplikasi yang bertujuan untuk mengingatkan atau memberitahukan tugas-tugas sekolah maupun kantor.
  Dengan adanya aplikasi ini diharapkan bisa membantu para siswa ataupun pegawai dari kelupaan tugas-tugas yang mereka miliki. Dengan
  aplikasi ini juga semua tugas yang telah dikerjakan akan tersusun dengan rapih dan dapat melihat tugas-tugas yang mereka kerjakan.
  
# Teknologi
  1. Java
  2. Firebase
  
# Fitur
  1. Login dengan google
  2. Register
  3. Dashboard
  4. Input tugas baru
  5. Lihat tugas yang belum selesai
  6. Lihat tugas yang sudah selesai
  7. Edit tugas
  8. Hapus tugas selesai
  9. Profile
